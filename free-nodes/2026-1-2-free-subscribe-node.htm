一、什么是Shadowrocket？为何它广受欢迎？

Shadowrocket 是一款 iOS 平台上的网络代理客户端，原生支持多种代理协议（如 Vmess、Shadowsocks、Trojan、HTTP、HTTPS、Socks5 等），以其丰富的自定义功能与高稳定性广受技术用户青睐。

它最大的优势体现在以下几个方面：

  • 精细化规则分流：可根据域名、IP、关键词制定个性化代理规则；

  • 多协议支持：让用户可以使用不同服务商的订阅链接；

  • 流量统计功能：可视化流量监控，了解每一个连接请求的去向；

  • 安全加密：通过高强度加密技术，实现端到端通信保护；

  • 自带测试工具：可测速、可ping、可查看连接日志，利于节点维护。

简而言之，它不是简单的“连上就能用”的工具，而是一个集代理控制、网络监控、DNS优化于一体的高级网络工具。

三、初始配置教程（文字 + 视频演示）

? 视频建议片段 02：【从配置链接导入节点，到规则设置全流程】

Shadowrocket 支持手动添加节点和导入订阅链接两种方式。

3.1 配置节点的两种方式

A. 手动添加代理节点（适用于少量节点使用）

  1. 打开 Shadowrocket；

  2. 点击右上角“+”号；

  3. 选择节点类型（如Shadowsocks、Vmess等）；

  4. 填写代理服务器的IP、端口、加密方式、UUID/密码；

  5. 命名该节点并保存。

B. 通过订阅链接导入（推荐方式）

  1. 获取你的代理服务商提供的 订阅链接（以 http/https 开头）；

  2. 在Shadowrocket主界面点击配置 > 订阅；

  3. 添加新订阅地址，输入链接后保存；

  4. 点击“更新”按钮获取所有节点；

  5. 返回首页，选择任意节点并连接即可。

四、规则设置与路由策略优化

? 视频建议片段 03：【规则演示：如何设置Netflix走代理、淘宝走直连】

Shadowrocket最强大的一项功能便是**“规则分流系统”**，这使得用户可以根据不同目的地URL决定是否走代理。

4.1 规则优先级说明（从高到低）

  1. DOMAIN-KEYWORD（关键词）

  2. DOMAIN（域名）

  3. IP-CIDR（IP段）

  4. FINAL（默认处理规则）

4.2 常见规则配置场景

  • Netflix / YouTube 全走代理：

    javascript
    DOMAIN-SUFFIX,netflix.com,Proxy
DOMAIN-SUFFIX,youtube.com,Proxy

  • 本地服务、支付宝、淘宝直连：

    pgsql
    DOMAIN-SUFFIX,alipay.com,DIRECT
DOMAIN-SUFFIX,taobao.com,DIRECT

  • 广告拦截规则（可添加AdBlock规则文件）

FAQ

QuantumultX 如何实现按应用分流？
QuantumultX 可以根据不同应用程序的流量类型设置分流规则，通过策略组选择最优节点。这样可保证游戏、视频和浏览器流量经过最优节点，提高访问速度和网络稳定性。
V2RayN 订阅更新后节点重复如何清理？
V2RayN 在导入订阅时有时会产生重复节点，可以点击右键菜单选择“删除重复项”。若仍存在旧节点缓存，可清空配置文件夹内的 history.json 并重新导入订阅，保证节点列表整洁。
WinXray 支持 Hysteria 协议吗？
新版本的 WinXray 已整合了 V2Ray 内核更新，支持 Hysteria 协议。用户可通过导入 Clash 或 sing-box 类型的订阅使用基于 QUIC 的高性能连接，速度和稳定性均更出色。
Mellow 的 TUN 模式能全局代理吗？
TUN 模式拦截系统 IP 层流量，可实现全局代理，无需手动设置浏览器或应用代理。适合需要全局科学上网或游戏加速的场景，保证所有流量均走代理节点。
Shadowrocket 节点延迟排序功能如何提升访问体验？
Shadowrocket 会测速节点延迟和丢包率，并优先使用低延迟节点。结合策略组和分流规则，可保证网页、视频和游戏流量通过最优节点，提高访问速度和稳定性。
Hysteria2 是否支持 UDP 加速？
支持。Hysteria2 通过 QUIC 协议原生支持 UDP 加速，对实时性要求高的应用（如游戏或语音通话）表现极佳，能显著降低延迟并提高连接稳定性。
Nekobox 可以定时更新订阅节点吗？
支持。用户可设置定时更新频率，例如每天或每小时，程序会自动下载订阅节点并替换旧节点，确保节点始终可用并保持最新线路信息。
如何防止租用服务器被用作滥发或代理滥用？
限制对外端口与协议，使用防火墙与入侵检测（如 fail2ban），监控流量异常并设置告警，定期审计登录行为，及时阻断滥用源并与服务商沟通封禁滥用账户以防引起供应商封禁整台机。
V2Ray 的 ws+tls 传输模式有什么优点？
ws+tls 模式利用 WebSocket 伪装成普通 HTTPS 流量，使得防火墙难以区分代理与正常访问。该模式兼容性强，适合部署在支持 Nginx 或 Caddy 的 Web 服务器上。
如何在 Clash Premium 中使用 TUN 模式加速游戏？
Clash Premium 版本支持系统级 TUN 模式，可将所有 UDP 流量转发至代理节点。启用该模式后需管理员权限，并在配置文件中添加 dns 与 tun 模块，选择低延迟节点即可实现游戏加速。
